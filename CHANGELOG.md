# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-19

### Added
- Initial release of Task Manager application
- User authentication (register/login) with JWT
- Task CRUD operations (Create, Read, Update, Delete)
- Task prioritization (Low, Medium, High)
- Task status tracking (To Do, In Progress, Completed)
- Due date management for tasks
- Task filtering by status
- Responsive UI with dark/light mode support
- RESTful API with Express.js
- MongoDB database integration
- TypeScript support for both frontend and backend
- React frontend with Vite
- State management with Zustand
- Form validation on frontend and backend
- Error handling middleware
- Request logging middleware
- Utility functions for task operations
- Utility functions for date formatting
- Search functionality components
- Task statistics dashboard
- Loading spinner component
- Error message component
- Comprehensive documentation (README, CONTRIBUTING, SECURITY)
- MIT License

### Technical Details
- Frontend: React 18, TypeScript, Vite, React Router, Zustand, Axios
- Backend: Node.js, Express, TypeScript, MongoDB, Mongoose, JWT, bcrypt
- Development tools: ESLint, TypeScript compiler
- Package managers: npm
